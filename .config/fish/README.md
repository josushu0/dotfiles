# Josushu0's Fish Shell configuration

![image](https://github.com/josushu0/dotfiles/assets/61262876/bb79830e-ea67-4410-9a91-df106554b2a6)

- [Fundle](https://github.com/danhper/fundle) as plugin manager
- Theme based on terminal colors
- [Krabby](https://github.com/yannjor/krabby) as a greeter to print random pokemon
- [Tide](https://github.com/IlanCosman/tide) as a prompt

## Requirements
- [fish](https://github.com/fish-shell/fish-shell)
- [krabby](https://github.com/yannjor/krabby)

## Fundle plugins
- [z](https://github.com/jethrokuan/z)
- [sponge](https://github.com/meaningful-ooo/sponge)
- [autopair](https://github.com/jorgebucaran/autopair.fish)
- [puffer-fish](https://github.com/nickeb96/puffer-fish)
- [tide](https://github.com/IlanCosman/tide)

**NOTE**: Plugins will autoinstall on fisrt launch
