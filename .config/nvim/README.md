# Josushu0's Neovim configuration

![Neovim in terminal](https://github.com/josushu0/dotfiles/assets/61262876/cd82c82d-7587-4b08-9fbc-a08bda635964)

Largely based on [LazyVim](https://github.com/LazyVim/LazyVim) with just some configuration on top.

## Requirements
- Neovim >= 0.9.0 (needs to be built with LuaJIT)
- Git >= 2.19.0 (for partial clones support)
- a Nerd Font (optional)
- a C compiler for nvim-treesitter. See here
